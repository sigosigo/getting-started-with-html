HTML

Html(otherwise known as Hypertext Markup Language) is the language used to make web pages.

Tags

Tags specify what each element is supppsed to do in html. Some tags have a begining and ending tag,such as bold, while others,
break for example, do not. Each tag is a command surrrounded by two angeled brackets(<>). Ending tags have a forward slash typed
inside the bracked before the command.
